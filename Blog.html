  • Question-1: What are the different ways to select an element in the DOM?

    let element = document.getElementById("myId"); , let elements = document.getElementsByClassName("myClass"); , let elements = document.getElementsByTagName("div"); , let elements = document.getElementsByName("myName"); , let element = document.querySelector(".myClass"); , let elements = document.querySelectorAll("div.myClass");

  • Question-2: What is the difference between innerHTML, innerText, and textContent ?

    innerHTML → Need HTML formatting , innerText → Only visible text , textContent → All text (better performance)

  • Question-3: What is event delegation in the DOM?

    Event Delegation is a technique where a parent element handles events for its child elements, even if they are dynamically added. This works due to event bubbling—events propagate from child to parent in the DOM.

  • Question-4: What is event bubbling in the DOM?

    Event Bubbling is a mechanism where an event starts from the target element and propagates up through its parent elements in the DOM.

  • Question-5: How do you create, add, and remove elements using JavaScript?

    createElement() → Creates an element. appendChild(), append(), insertBefore() → Add elements. remove(), removeChild() → Remove elements.
